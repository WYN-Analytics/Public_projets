

# Power BI Certification Project – Call Center Performance Analysis

## Contexte du projet

Dans le cadre de ma formation Data Analyst, j’ai réalisé un examen de certification Power BI basé 
sur des données réelles anonymisées d’une entreprise de centre d’appels externalisé.

L’objectif était de concevoir un rapport décisionnel destiné à un directeur régional, permettant d’analyser :

    La performance du service client

    L’évolution des revenus générés par les appels

    La performance des managers et des équipes

Ce projet mobilise l’ensemble du cycle d’analyse dans Power BI :
Data exploration → Data cleaning → Data modeling → DAX → Data visualization → Storytelling décisionnel.

### Analyse comportementale :

- Performance individuelle des managers  
- Performance par équipe  
- Temps d’attente moyen par employé  
- Volume d’appels traité  
- Contribution au revenu global  

---

##  Conception du Dashboard

Le rapport est structuré pour un directeur régional :

- Page synthèse avec KPIs principaux  
- Pages détaillées par axe d’analyse  
- Filtres dynamiques (année, site, type, manager)  
- Visualisations orientées décision  
- Données non pertinentes masquées  

**Objectif :** fournir une lecture rapide et stratégique de la performance.
